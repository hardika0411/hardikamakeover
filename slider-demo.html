Image Slider Demo

This is a demonstration of the sliding image feature implemented for Hardika Makeover website. The slider includes automatic slideshow, manual navigation, touch/swipe support, and keyboard controls.

Slider Features:

  • Auto-play slideshow (changes every 5 seconds)
  • Click navigation arrows to manually control slides
  • Click dots to jump to specific slides
  • Touch/swipe support for mobile devices
  • Keyboard navigation (Arrow keys, Spacebar, Escape)
  • Auto-pause when hovering over the slider
  • Smooth transitions and animations
  • Fully responsive design
